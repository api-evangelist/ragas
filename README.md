# Ragas (ragas)

Ragas is an open-source Python evaluation framework for LLM and RAG (retrieval-augmented generation) applications. It provides metrics, experiments, dataset management, and integrations with LangChain and LlamaIndex. Ragas is delivered as a Python library, not a hosted REST API.

## APIs

Ragas is an open-source Python library (Apache 2.0). It runs in your own process. There is no hosted REST API, no API key, and no per-call charge from the Ragas project itself. Cost arises only from the underlying LLM/embedding model providers (OpenAI, Anthropic, etc.) that Ragas calls during evaluation.
